My Weather Data Website


The way the whole analysis was made was by retrieving, manipulating and analyzing an open-to-public API called "Weather API".

Firstly, an account was created in the Weather API website. Once it was done, the website provided a unique API Key used to get access to the information and manipulate it. The tool used to make the analysis was a Jupyter Notebook along with the Pandas library using Python as the coding language. The idea was to plan a ficticious holiday trip in such a way that parameters where needed to enter into the anlysis to get the best fits for the perfect holiday trip.

Some of the parameters entered into the code were:

  • Minimum temperature
  • Maximum temperature
  • Latitude
  • Longitude
  • Precipitation
  • Humidity

Google Places was used for the second part of the assignment. The idea was that once all of the locations were found that fitted the paramters entered, a list of hotels would display in those resulting cities with hotel data such as the Hotel Name and the Hotel Rating.

Finally, a world map would be displayed with all of those cities pointed out with markers that once the user hover on those markers, a message would appear to the user showing get the City Name and Hotel information.
