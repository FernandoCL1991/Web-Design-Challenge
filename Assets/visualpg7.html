The Weather Data Site



The way the whole analysis was made was by retrieving, manipulating and analyzing an open-to-public API called "Weather API".

Firstly, an account was created in the Weather API website. Once it was done, the website provided a unique API Key used to get access to the information and manipulate it.

The tool used to make the analysis was a Jupyter Notebook along with the Pandas library using Python as the coding language.

The idea was to plan a ficticious holiday trip in such a way that parameters where needed to enter into the anlysis to get the best fits for the perfect holiday trip.

Some of the parameters entered into the code were:


  • Minimum temperature
  • Maximum temperature
  • Latitude
  • Longitude
  • Precipitation
  • Humidity
